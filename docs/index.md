# 蜀桥纪事

## 项目介绍

蜀地素有“后方粮仓”之美称，其中的往来经济贸易络绎不绝，而作为重要交通枢纽的桥梁也见证着古蜀地政治经济繁荣发展。古桥兼具观赏性和实用性，不仅体现着古代劳动人民的非凡智慧与才能，其中蕴含的科技、文化等多方面价值也是不可忽视的。本项目从已有的现存蜀地古桥资料入手，在历史、风土人情、政治经济以及社会发展等多个方面对其进行了阐述。
